Windows 系统安装教程

发表于 2018-03-03 更新于 2023-06-07

一、概述

###1.安装系统前的准备工作
###2.安装系统的步骤
###3.安装完成后安装驱动和软件

二、具体过程

1.安装系统前的准备工作

需要准备的有：
  • 迅雷软件
    使用迅雷软件下载系统镜像文件。
  • U盘启动盘制作工具
    制作U盘启动盘
  • U盘
    制作成启动盘后进入PE系统装机
  • Windows10系统镜像文件
    需要安装的系统文件
  • 一台可以正常运行的Windows电脑
    制作U盘启动盘
准备阶段需要完成的操作：

1.打开系统镜像下载地址，选择合适的系统镜像进行下载。

Tips：根据需要选择系统类型（Windows 7 或者 Windows 10）
Tips：不同机器选择 x86 或者 x64 系统进行下载，如果你不清楚两种的区别，一般情况下 x64 系统是适合大多数机器的。

选中并复制其中的 ed2k 开头的磁力链接，到迅雷软件中进行下载。（如果速度缓慢可以在网上寻找迅雷临时共享会员，提升速度）。

2.系统镜像下载的同时，可以着手准备制作U盘启动盘了

准备好U盘，启动盘制作工具和一台可用的电脑：
U盘容量不是特别小都可以使用（一般的U盘都可以胜任）；启动盘制作工具可以百度任何一款工具，供选择的很多，如果是小白用户，可以使用

  • 老毛桃
  • 大白菜
    启动盘制作工具。

选中当前要用的U盘，然后使用点击一键制作USB启动盘；制作完成后还可以进行模拟启动，如果可以正常模拟启动就制作好了启动U盘。

*Tips：*制作过程中会格式化U盘内所有数据，如果有重要数据需要提前进行转移或备份。

3.将下载完成的系统镜像文件保存到U盘上
如果需要将电脑磁盘全部数据进行格式化，就要将系统镜像文件放到外部存储设备中，在后续系统安装过程中使用。

2.安装系统的步骤

1.重启电脑，进入BIOS进行设置：

在电脑上插入刚刚制作好的U盘启动盘，然后重启电脑，在启动过程中按下键盘上的 F2、F12、或者 Delete 键，进入电脑 BIOS 系统。
之后在 BIOS 中 Boot 栏，调整 Boot 启动顺序 ，将制作好的启动盘作为第一启动顺序。选择好之后，安F10 保存对 BIOS 的设置，重启进入 PE系统。

*Tips：*进入 BIOS 的按键，不同电脑设置不同，可以自行百度具体机型对应按键。
*Tips：*因不同电脑 BIOS 界面不同，如果确实无法自行完成，请自行搜索对应型号设置方法。

2.进入 Windows PE 系统进行系统安装：

在设置好 BIOS 启动之后可以进入 PE 选项，如下图：

稍等一会儿就可以进入 PE 系统了：

进入系统以后 ,选择磁盘分区工具：

在磁盘工具中可以对磁盘进行分区、格式化、建立磁盘引导等操作。

首先在磁盘工具中选中需要做系统盘的磁盘，点击建立分区–>主分区
–>设置主分区激活状态。然后可以对系统内其他磁盘空间新建逻辑分区，将所有分区全部设置完成后，点击保存。软件会提示是否需要进行格式化操作，此时选择格式化磁盘。完成后退出分区工具。

选中系统安装工具：

此时可以将系统镜像文件拷贝到电脑磁盘上，并且进行解压。

在选中系统源文件的输入框中，点击右侧选择然后选中之前下载的解压以后的系统镜像文件，找到其中的 sources 文件夹下的 install.wim 文件，选中打开。
在系统安装工具中配置其他选项，其中的 系统版本可以选择Windows 10 专业版，其余的设置可以使用默认。点击开始安装，安装工具就会开始部署文件到磁盘上。

部署完成后，弹出U盘，重启电脑，系统会自动进入安装程序：

等待一会儿：
完成一些基础的设置，就可以看到安装完成后的操作系统桌面了。

至此Windows 10 系统重装完成。

3.安装完成后安装驱动和软件

一般 Windows 10 系统都会自带WiFi 联网驱动，建议联网后使用360驱动大师一键安装设备驱动程序。
